# Dataset Description

## Overview

This dataset contains rendered cryo-electron microscopy grid-square inspection images. Each scene shows a grid square with circular holes, variable ice thickness, contamination, cracks, fiducial beads, frost, low-dose noise, carbon shadow, beam gradients, and other inspection artifacts.

The raw data is generated by the included deterministic script `generate_raw.py`. The generator records a hidden scene trace and derives multiple-choice visual question answers from that trace. The public task requires reading the image and question together; trace columns are removed during preparation.

This is a visual question answering dataset, not a single-label image classifier. Several questions can refer to the same image, and each question asks about a different visual property.

## File Structure

| File or Folder | Rows / Files | Description |
|---|---:|---|
| `data.csv` | 2,460 rows | One row per visual question with raw IDs, image path, question, choices, answer label, hidden groups, and trace fields. |
| `images/` | 820 PNG files | Rendered 160x160 RGB cryo-EM grid-square inspection images. |
| `generate_raw.py` | 1 file | Deterministic generator with fixed random seed. |

The prepare script remaps raw IDs to opaque IDs, copies images to public hashed filenames, performs a scene-level train/test split, removes hidden trace columns, and creates `train.csv`, `test.csv`, `sample_submission.csv`, and private `answers.csv`.

## Raw Columns

| Column | Type | Description |
|---|---:|---|
| `question_id` | string | Raw question identifier. |
| `scene_id` | string | Raw image/scene identifier. |
| `image_path` | string | Relative path to the PNG grid-square image. |
| `question_type` | string | Visual skill being tested. |
| `question` | string | Multiple-choice question text. |
| `choice_a` to `choice_d` | string | Answer choices. |
| `answer_label` | string | Correct answer: A, B, C, or D. |
| `difficulty` | string | Hidden group: easy, medium, or hard. |
| `visibility` | string | Hidden condition: clear, frosty, low_dose, or crowded. |
| `layout_family` | string | Hidden grid layout family. |
| `ood_axis` | string | Hidden shift such as carbon_shadow, beam_gradient, or ice_rings. |
| `trace_*` | mixed | Generator trace columns used to derive answers. Removed from public files. |

## Question Types

| Type | What It Tests |
|---|---|
| `best_quadrant` | Find the quadrant with the most usable thin-or-medium ice holes. |
| `contamination_quadrant` | Locate the strongest contamination cluster. |
| `usable_hole_count` | Count usable holes in binned ranges. |
| `crack_orientation` | Identify no crack, horizontal, vertical, or diagonal crack direction. |
| `ice_thickness` | Identify dominant ice thickness: thin, medium, thick, or mixed. |
| `fiducial_count` | Count bright gold fiducial beads. |
